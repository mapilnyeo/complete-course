# 커피 성격 퀴즈 - 요구사항 (Requirements)

## 🛠️ 기술 스택
- **Framework:** Next.js (최신 버전)
- **Styling:** Vanilla CSS 또는 Tailwind CSS (깔끔한 UI 지향)
- **Deployment:** Vercel

## 🏗️ 퀴즈 구조
- **질문 개수:** 총 5개
- **선택지:** 각 질문당 4개 (A, B, C, D)
- **점수 방식:** 가장 많이 선택된 답변 유형에 따라 결과 도출

## 🎯 성격 유형 및 커피 페어링
1. **에스프레소 탐험가 (대부분 B):** 분석적이고 본질에 집중하는 타입 ➡️ **에스프레소 / 리스트레토**
2. **라떼 러버 (대부분 C):** 부드럽고 안정적인 것을 선호하는 타입 ➡️ **바닐라 라떼 / 카페 모카**
3. **콜드 브루 어드벤처러 (대부분 A):** 모험적이고 활동적인 타입 ➡️ **나이트로 콜드 브루**
4. **카푸치노 클래식 (대부분 D):** 사람과 어울리는 것을 좋아하는 소셜 타입 ➡️ **카푸치노 / 피콜로**

## 📝 퀴즈 질문들
1. **아침의 시작:** 상쾌한 아침, 당신은 어떻게 시작하나요?
2. **결정 스타일:** 중요한 결정을 내려야 할 때, 당신은?
3. **완벽한 주말:** 가장 이상적인 토요일 오후는?
4. **스트레스 대처:** 일이 너무 많아 압도될 때, 당신은?
5. **카페에서의 주문:** 처음 가보는 카페에 들어갔을 때, 당신은?

## 🎨 시각적 스타일
- **톤:** 따뜻하고 친근하며 공동체 중심적인 베이스캠프 브랜드 이미지 반영
- **모드:** 라이트 모드 (따뜻한 크림색과 커피 톤의 브라운 조합)
- **이미지:** 결과 페이지에 각 커피 음료의 AI 생성 이미지 포함

## ✨ 결과 페이지 구성
- 커피 성격 이름 및 추천 음료 명시
- 성격에 대한 따뜻한 설명 (3-4문장)
- 다시 하기 버튼 및 결과 공유 기능(모의) 포함
