## GEMINI.md 계층 구조

Gemini CLI는 계층적으로 GEMINI.md 파일을 읽어요:

1. **글로벌 GEMINI.md** - `~/.gemini/GEMINI.md` - 모든 프로젝트에서 항상 적용
2. **프로젝트 GEMINI.md** - 현재 폴더나 상위 폴더 - 이 프로젝트에만 적용

그래서 "항상 한국어로 대답해" 같은 개인적 선호사항을 글로벌 GEMINI.md에 넣고, Basecamp Coffee 컨텍스트는 프로젝트 GEMINI.md에 넣을 수 있어요.

## /memory 명령어

GEMINI.md에 접근하는 것 외에도, Gemini CLI에는 메모리 관리를 위한 `/memory` 명령어가 있어요:

- `/memory show` - 현재 로드된 모든 메모리 보기
- `/memory add "당신이 추가하고 싶은 것"` - 프로젝트 GEMINI.md에 직접 추가
- `/memory refresh` - GEMINI.md 파일을 수동으로 편집한 경우 컨텍스트 다시 로드
